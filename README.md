## 🎨 Customization Guide

### Colors
The primary accent color is `#39e6b5` (mint green). To change it, update the color classes in components and `globals.css`.

### Content
Update the content in each section component:

| Component | Content |
|-----------|---------|
| `Hero.tsx` | Name, title, bio, tags, CTA buttons |
| `About.tsx` | About me description, stats |
| `Skills.tsx` | Skills categories and technologies |
| `Experience.tsx` | Work experience timeline |
| `Projects.tsx` | Project showcase |
| `Contact.tsx` | Contact information, social links |

### Font
Poppins font is loaded via Next.js Font Optimization. To change the font, update `app/layout.tsx`.
